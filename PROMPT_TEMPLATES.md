# PROMPT_TEMPLATES.md

Reusable prompts for this repository.

## 1) Implement a feature
```
Read AGENTS.md and PROJECT_CONTEXT.md first.
Implement: <feature>.
Constraints: <constraints>.
Validation: run relevant tests.
Update CHANGELOG.md with an Unreleased entry.
```

## 2) Fix a bug
```
Read AGENTS.md and PROJECT_CONTEXT.md first.
Bug: <symptom>.
Expected behavior: <expected>.
Please find root cause, implement a minimal fix, and validate with tests.
Update CHANGELOG.md.
```

## 3) Update documentation
```
Sync README.md, PROJECT_CONTEXT.md, AGENTS.md, and CHANGELOG.md with current behavior.
Keep wording concise and accurate.
```

## 4) New machine / fresh chat onboarding
```
Read README.md, AGENTS.md, PROJECT_CONTEXT.md, and CHANGELOG.md.
Then summarize:
1) what this project does
2) key architecture and data flow
3) known risks
4) recommended next steps
```
